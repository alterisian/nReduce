== nReduce Rails App

App Requirements:

>> RVM
* rvm install ruby-1.9.3-p125-falcon
* rvm use ruby-1.9.3-p125-falcon
* rvm create gemset nreduce-web
>> Redis
>> MySQL

== Testing

To run full test suite: bundle exec guard start

== Resque

To start a worker on all queues run this: RAILS_ENV=production QUEUE=* bundle exec rake environment resque:work

== Whenever - for periodic emails - write to crontab

bundle exec whenever -w
